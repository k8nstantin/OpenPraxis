How It Works
1
Memories are shared

Every Claude session on this machine reads and writes to the same memory. What one session learns, all sessions know.

2
Conversations are captured

Every interaction across all sessions is recorded automatically. Browse them in the Conversations tab.

3
Copy & paste references

Click the ⎘ button on any memory or conversation turn to copy a reference. Paste it into Claude and it will recall the exact content.

4
Talk to Claude naturally

"Save to memory: the catalog name must equal the bucket name" — stores it.
"What do we know about catalog naming?" — searches for it.
"Recall memory 69425f8e" — fetches a specific memory by its marker.
